## 🚀 Quick Start

### Prerequisites
- PHP 7.4 or higher
- MySQL database
- Web server (Apache/Nginx)
- cPanel hosting (recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/cyrus2005/personalPortfolioCyrus.git
   cd personalPortfolioCyrus
   ```

2. **Upload to your web server**
   - Upload all files to your `public_html` directory
   - Ensure proper file permissions are set

3. **Configure email settings**
   - Edit `includes/contact_handler.php`
   - Update the email address on line 15

4. **Set file permissions**
   ```bash
   chmod 755 logs/
   chmod 644 includes/contact_handler.php
   ```

5. **Test the website**
   - Visit your domain
   - Test the contact form
   - Verify all portfolio links work

## ⚙️ Configuration

### Email Setup
Update the contact form handler to use your email:
```php
// In includes/contact_handler.php
$to = 'your-email@yourdomain.com'; // Line 15
```

### Customization
- **Colors**: Update CSS variables in `assets/css/style.css`
- **Content**: Modify portfolio projects and services
- **Contact Info**: Update phone and email throughout the site
- **Images**: Replace placeholder images with actual screenshots
